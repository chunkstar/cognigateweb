Frequently Asked Questions

What's the difference between Indie and Platform tiers?
Indie is for developers tracking their own AI costs across projects. Platform is for developers building AI products (chatbots, SaaS) who need to track costs PER END-USER and set tier-based limits (free users get 10 msgs/day, pro users get unlimited).
What counts as an "end-user" in Platform tier?
An end-user is someone using YOUR application that makes AI requests through Cognigate. For example, if you built an AI chatbot with 500 users, those 500 users would each count as one end-user. The Platform tier includes 1,000 end-users, then $0.10/user/month after that.
Do I need to migrate my API keys to Cognigate?
No! Cognigate is a proxy - you keep using your own API keys. We just route requests through our system to track usage, enforce limits, and provide analytics. Your keys stay secure and never leave your control.
What happens if I hit my budget limit?
You set a daily budget (e.g., $50/day). When you hit 80%, you get an email alert. At 100%, Cognigate automatically stops routing requests to prevent overages. You can also set it to fallback to local models (Ollama) instead of stopping entirely.
Can I try Platform tier before paying?
Yes! All paid tiers include a 14-day free trial. No credit card required. You can test per-user tracking, tier limits, and all features before committing.
How does the 20% annual discount work?
Pay annually and save 20%. For example: Indie is $29/mo ($348/year) or $278/year if paid annually. That's $70 in savings! You can switch between monthly and annual at any time.
